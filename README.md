# Meta Ads Performance Dashboard (Power BI)

## Project Overview

The **Meta Ads Performance Dashboard** is an interactive analytics solution built using Power BI to monitor and evaluate the performance of digital advertising campaigns across Meta platforms such as Facebook and Instagram.

The dashboard provides insights into campaign performance, engagement trends, audience demographics, and geographic reach. It enables marketers and analysts to quickly identify high-performing campaigns, track key performance indicators (KPIs), and optimize advertising strategies.

---

## Objectives

* Analyze advertising performance across Meta platforms
* Track engagement and conversion metrics
* Identify audience demographics and behavioral patterns
* Monitor campaign performance over time
* Provide actionable insights for marketing optimization

## Key Performance Indicators (KPIs)

The dashboard tracks the following metrics:

* Total Impressions
* Total Clicks
* Shares
* Comments
* Purchases
* Engagement
* Click Through Rate (CTR)
* Engagement Rate
* Conversion Rate
* Purchase Rate
* Total Budget
* Average Budget Per Campaign

---

## Dashboard Features

### 1. KPI Summary Section

Provides a quick overview of major campaign performance indicators including impressions, clicks, engagement, and purchases.

### 2. Audience Demographics Analysis

Displays engagement by:

* Gender
* Age groups

This helps identify the most responsive audience segments.

### 3. Engagement Trend Analysis

Tracks weekly and hourly engagement trends to understand when audiences interact most with advertisements.

### 4. Geographic Performance Analysis

Visualizes engagement by geographic regions using an interactive map to understand global campaign reach.

### 5. Campaign Performance Analysis

Analyzes campaign performance by:

* Ad Type
* Campaign Name
* Target Interest

### 6. Interactive Filters

Users can dynamically filter the dashboard by:

* Platform (Facebook / Instagram)
* Campaign Name
* Target Interest
* Selected Metrics

---

## Key Insights Derived

* Certain ad formats generate significantly higher engagement.
* Engagement varies across different audience age groups.
* Geographic regions respond differently to campaigns.
* Campaign timing affects engagement levels.
* CTR and Conversion Rate provide insights into campaign effectiveness.
